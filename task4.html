Welcome To THE HUMAN COCKROACH GAME


INSTRUCTION FOR PLAYING THE GAME:

  1. The user will be allowed to get a computer generated input By pressing the GENERATE RANDOM INPUT button.
  2. The mapping has been done in following way:{input%3 will be generated}
  3. If remainder is 0,Then it is HUMAN. If remainder is 1,Then it is COCKROACH and If remainder is 2,Then it is NUCLEAR BOMB.
  4. The number should be in range 0-1000
  5. If Object 1 is Human, and object 2 is Cockroach; then the Human Survives
  6. If Object 1 is Cockroach, and object 2 is Nuclear Bomb, then Cockroach survives
  7. If Object 1 is Human, and Object 2 is Nuclear Bomb, then the Human dies.
